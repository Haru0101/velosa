    # @font-faceの書き方

    # 基本的な記述

    @font-face {
    font-family: 'フォント名';
    src: url('フォントファイルがあるパス') format('フォントの形式');
}

    最低限、これだけ記述していれば表示されます。
    urlの部分は、今回ではローカルのパスを指定しますが、WEBフォントを使う場合は、そのURLを指定することもできます。
